# Unterschiede zwischen Copilot Ask, Edit und Agent Modus

## Copilot Ask
- **Zweck:** Beantwortet Fragen zu Code, Technologien oder Programmierung.
- **Interaktion:** Liefert textbasierte Antworten, Erklärungen oder Beispiele.
- **Typische Nutzung:** Verständnisfragen, Dokumentation, Code-Snippets.

## Copilot Edit
- **Zweck:** Führt gezielte Änderungen am bestehenden Code durch.
- **Interaktion:** Nimmt konkrete Anweisungen entgegen und generiert Code-Änderungen.
- **Typische Nutzung:** Refactoring, Bugfixes, Hinzufügen/Entfernen von Funktionen.

## Copilot Agent
- **Zweck:** Führt komplexe Aufgaben aus, die mehrere Schritte oder Dateien betreffen.
- **Interaktion:** Arbeitet als KI-Agent, kann Aufgaben planen und ausführen.
- **Typische Nutzung:** Automatisierte Workflows, größere Code-Transformationen, Multi-File-Änderungen.

## Zusammenfassung
- **Ask:** Für Fragen und Erklärungen.
- **Edit:** Für gezielte Code-Änderungen.
- **Agent:** Für komplexe, mehrschrittige Aufgaben und Automatisierungen.
